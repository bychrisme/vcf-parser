# VCF Parser

Use this API parse your VCF contact on JSON.

## Tech

* Node Js

## Installation

Clone the repository

```
$ git clone https://github.com/bychrisme/vcf-parser.git [project_name]

```

Install dependencies

```
$ cd [project_name]
```
```
$ npm i
```

Now you start

```
$ cp .env.example .env

$ npm start
```
